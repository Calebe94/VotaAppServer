# VotaAppServer

Web service para o [VotaApp](https://github.com/thecobra159/VotaAPP/tree/develop).

## Setup

* Instalar as Dependências:
  ```
    npm install
  ```

* Para criar o banco:
  ```
    npm run createdb
  ```

* Rodar o servidor para testes:
  ```
    npm test
  ```

* Rodar o servidor para Produção:
  ```
    npm start
  ```

## Banco

* Nome do **Database** 
  * vota_app

* Collections
  * candidatos
  * eleitores
